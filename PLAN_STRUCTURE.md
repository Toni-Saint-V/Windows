## 2. Что убрать
- все тексты про концепт, брендбук и сам лендинг
- англоязычные лейблы интерфейса
- любые абстрактные объяснения, не связанные с продажей
- любые внешние стоковые изображения

## 3. Что наполнить реальным контентом
- оффер, преимущества, порядок работы, отзывы, контакты — только по `glavzaves-okna.ru`
- галерея — только присланные реальные фотографии
- подписи к кейсам — только по визуально подтвержденным объектам
- CTA — единый и короткий copy без лишних формулировок

## 4. Две ключевые фичи
### Smart Calculator
- отдельный блок
- пользователь задает тип объекта, размер, сезонность, монтаж
- на выходе получает ориентировочный диапазон цены или пометку `точный расчет после замера`
- CTA: отправить расчет в Telegram

### Smart Object Advisor
- отдельный блок, не смешивать с калькулятором
- пользователь отвечает на короткие вопросы по объекту и приоритету
- на выходе получает рекомендованный сценарий решения и список рекомендуемых опций
- CTA: отправить подбор в Telegram

## 5. Что доводит сайт до 100%
- Telegram становится основным каналом по всему сайту
- `index.html` готов для деплоя
- все изображения локальные и реальные
- семантика чистая: `header`, `nav`, `main`, `section`, `footer`, `address`, `ol`
- mobile-first проверка по всем секциям
- все CTA, формы и ссылки работают
- мета-теги, favicon, OG и финальный polish доводятся перед публикацией
